# Trade integration in Europe in the presence of unobservable barriers

The folder include all replication package, containing the DTA and Do files with the dataset and scripts needed to obtain the main results shown in the research article. The files are ready to be used in STATA.

8 Files are included. 3 do.files that contain the code to replicate tables and figures of the paper and 5 dta files, including the region-to-region database (Data) in a zip file.
•	Baseline_models.do
•	Figures.do
•	Robustness.do
•	Data.dta
•	Data_maps.dta
•	NUTS_RG_01M_2016_4326_LEVL_2.dta
•	NUTS_RG_01M_2016_4326_LEVL_2.shp
•	NUTS_RG_01M_2016_4326_LEVL_2_shp.dta
